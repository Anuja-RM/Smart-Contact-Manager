About

Smart Contact Manager is a secure and user-friendly web application that helps users store, organize, and manage their contacts efficiently. With features like contact editing, personalized dashboards, and dark/light theme compatibility, it ensures a modern and accessible experience.

Why This Project?

Managing contacts is an everyday need, whether personal or professional. This project aims to provide an easy-to-use digital solution that reduces dependency on manual records and keeps everything in one place.

Technologies Used

Spring Boot
Hibernate & JPA
Tailwind + Thymeleaf
MySQL + Spring Security

Our Vision

To build a simple yet powerful platform that demonstrates the integration of modern web technologies while solving a practical problem.
